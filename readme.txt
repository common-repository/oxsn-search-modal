This plugin adds helpful search modal shortcodes with quicktags!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/oxsn-search-modal` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->OXSN Search Modal screen to configure the plugin
